# Todo App

A simple interactive todo list web app.

## Features

- Add new tasks
- Mark tasks complete
- Edit tasks
- Delete tasks
- Filter tasks by All, Active, and Completed

## Usage

1. Open `index.html` in a browser.
2. Type a task into the input field.
3. Click the Add button or press Enter.
4. Use the filter buttons to view tasks.

## Notes

- The app is styled for both desktop and mobile screens.
- The layout adapts with responsive spacing and controls.
